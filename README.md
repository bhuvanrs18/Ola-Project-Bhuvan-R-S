OLA-Booking-Data-Analysis-and-Dashboard

📊 Data Analysis and Dashboard using OLA Booking Dataset with Python and Power BI

📌 Project Overview

This project focuses on analyzing OLA booking data to extract meaningful business insights related to ride bookings, cancellations, revenue generation, customer behavior, and payment trends.

The analysis is performed using Python, and the insights are visualized through an interactive Power BI dashboard.

🎯 Objectives
Analyze booking and revenue trends
Identify successful and cancelled ride patterns
Understand customer and driver cancellation behavior
Analyze payment method usage
Identify peak booking periods and ride demand
Support data-driven business decisions
📂 Dataset
Name: OLA Booking Dataset
Type: Structured CSV Data
Features: Booking ID, Customer ID, Vehicle Type, Ride Distance, Booking Status, Payment Method, Cancellation Reason, Revenue, Ratings, Date & Time
🛠️ Tech Stack
Python (Jupyter Notebook)
Pandas, NumPy
Matplotlib, Seaborn
Power BI

📈 Key Insights
Most bookings are completed successfully

This project demonstrates how raw booking data can be transformed into actionable business insights using data analysis and visualization tools.
